# 资料管理系统2.1 - 功能详解

## 一、核心功能

### 1. 工作目录管理
- **工作目录模式**：选择一个文件夹作为主工作目录，直接读取文件系统
- **链接外部文件夹**：可以链接其他位置的文件夹到侧边栏，方便统一管理
- **目录树**：VSCode风格的目录树，支持展开/折叠，带连线显示层级关系
- **自动收缩**：导航时自动收缩未浏览的文件夹（可在设置中关闭）

### 2. 文件浏览与预览
- **单击预览**：单击文件即可在右侧面板预览内容
- **双击打开**：双击文件用系统默认程序打开
- **多标签页**：支持在新标签页中打开文件夹（右键菜单）
- **面包屑导航**：顶部显示当前路径，可点击跳转
- **返回按钮**：快速返回上一层目录

### 3. 文件类型支持
支持预览的文件类型：
- **文本文件**：txt, md, log, csv, json, xml, yaml, ini, cfg
- **代码文件**：py, js, ts, java, c, cpp, h, go, rs, rb, php, html, css, vue, svelte
- **图片文件**：png, jpg, jpeg, gif, bmp, webp, svg, ico, tiff
- **其他**：显示文件类型图标和语言标识

### 4. 文件类型图标系统
每种文件类型都有专属emoji图标：
- 🐍 Python | 📜 JavaScript | ☕ Java | ⚙️ C/C++ | 🌐 HTML | 🎨 CSS
- 📄 文档 | 📊 表格 | 📦 压缩包 | 🎵 音频 | 🎬 视频 | 🖼️ 图片

### 5. 标签与备注系统
- **彩色标签**：为文件添加彩色标签，支持多个标签（逗号分隔）
- **备注功能**：为文件添加文字备注
- **标签颜色**：基于标签名自动生成15种颜色之一
- **实时显示**：标签在文件列表中实时显示

### 6. 智能搜索
#### 基础搜索
- 输入关键词搜索文件名
- 支持模糊匹配

#### 高级筛选语法
| 语法 | 功能 | 示例 |
|------|------|------|
| `#标签` | 按标签筛选 | `#重要` |
| `!扩展名` | 按文件类型筛选 | `!py` `!image` `!code` |
| `@备注` | 按备注内容筛选 | `@待处理` |
| `yaml:key` | YAML标签筛选 | `yaml:category` |
| `-#标签` | 排除标签 | `-#废弃` |
| `-!扩展名` | 排除类型 | `-!exe` |
| `空格` | AND逻辑（同时满足） | `#重要 !py` |
| `\|` | OR逻辑（满足其一） | `#重要\|#紧急` |

#### 快捷类型分组
- `!image` → png, jpg, gif, bmp, webp, svg, ico
- `!code` → py, js, ts, java, c, cpp, go, rs, rb, php
- `!doc` → doc, docx, pdf, txt, md

#### 中文符号支持
全角符号自动转换：`＃`→`#` `！`→`!` `＠`→`@` `｜`→`\|` `：`→`:`

#### 路径跳转
在搜索框输入Windows路径（如 `C:\Users\admin`）自动跳转到该目录

### 7. 右键菜单系统
#### 文件右键菜单
- 打开文件
- 在资源管理器中打开
- 复制路径/文件名
- 备份到同目录
- 复制/剪切/粘贴
- 重命名/删除

#### 文件夹右键菜单
- 打开文件夹
- 在新标签页打开
- 在新窗口打开
- 复制路径
- 粘贴到此文件夹
- 重命名/删除

#### 空白处右键菜单
- 刷新
- 新建文件夹
- 从资源管理器粘贴
- 在资源管理器中打开

#### 预览区右键菜单
- 复制选中文本
- 全选

#### 标签/备注输入框右键菜单
- 剪切/复制/粘贴
- 全选/删除

### 8. 文件操作
- **复制/剪切/粘贴**：支持应用内和系统剪切板
- **备份**：一键备份文件到同目录（自动添加时间戳）
- **重命名**：支持重命名文件和文件夹
- **删除**：删除文件或文件夹
- **拖拽**：从App拖文件到资源管理器
- **键盘快捷键**：
  - `Ctrl+C` 复制
  - `Ctrl+X` 剪切
  - `Ctrl+V` 粘贴
  - `Ctrl+A` 全选
  - `Ctrl+N` 新建文件夹
  - `F2` 重命名
  - `F5` 刷新
  - `Delete` 删除
  - `Backspace` 返回上一层
  - `Enter` 打开选中项
  - `Esc` 关闭面板

### 9. 主题系统
- **暗色主题**：深色渐变背景，护眼
- **亮色主题**：浅色背景，清晰
- **一键切换**：侧边栏底部快速切换
- **记住选择**：主题设置自动保存

### 10. 多窗口模式
- **标签页**：支持在新标签页中打开文件夹
- **多窗口**：支持在新窗口中打开文件夹
- **设置开关**：可在设置中开启/关闭

- **元数据**：`%APPDATA%/file-database-manager/metadata.json`

- **跨版本兼容**：配置格式向前兼容，2.0配置可导入2.1

## 三、REST API服务

### 概述
内置HTTP API服务器，端口默认5000，供AI Agent调用。

### API端点

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/health` | 健康检查 |
| GET | `/api/workspace` | 工作目录信息 |
| GET | `/api/files` | 列出文件 |
| GET | `/api/files/*` | 读取文件内容 |
| GET | `/api/stats` | 统计信息 |
| GET | `/api/docs` | API文档 |
| POST | `/api/search` | 搜索文件 |
| POST | `/api/meta` | 获取标签/备注 |
| PUT | `/api/meta` | 保存标签/备注 |
| POST | `/api/create-folder` | 创建文件夹 |
| POST | `/api/delete` | 删除文件 |
| POST | `/api/rename` | 重命名 |

### 调用示例

```bash
# 健康检查
curl http://127.0.0.1:5000/api/health

# 获取工作目录信息
curl http://127.0.0.1:5000/api/workspace

# 列出文件
curl http://127.0.0.1:5000/api/files?path=src&depth=1

# 搜索文件
curl -X POST http://127.0.0.1:5000/api/search \
  -H "Content-Type: application/json" \
  -d '{"keyword": "main.js"}'

# 读取文件内容
curl http://127.0.0.1:5000/api/files/src/main.js

# 给文件打标签
curl -X PUT http://127.0.0.1:5000/api/meta \
  -H "Content-Type: application/json" \
  -d '{"path": "src/main.js", "tags": "重要,核心", "notes": "主入口文件"}'

# 获取文件标签
curl -X POST http://127.0.0.1:5000/api/meta \
  -H "Content-Type: application/json" \
  -d '{"path": "src/main.js"}'

# 创建文件夹
curl -X POST http://127.0.0.1:5000/api/create-folder \
  -H "Content-Type: application/json" \
  -d '{"name": "new-folder"}'

# 删除文件
curl -X POST http://127.0.0.1:5000/api/delete \
  -H "Content-Type: application/json" \
  -d '{"path": "old-file.txt"}'

# 重命名
curl -X POST http://127.0.0.1:5000/api/rename \
  -H "Content-Type: application/json" \
  -d '{"path": "old-name.txt", "newName": "new-name.txt"}'
```

### AI Agent集成
本软件可与以下AI Agent配合使用：
- **MiMo Code**：通过API读取项目文件，辅助代码开发
- **Claude Code**：通过API分析代码结构，提供优化建议
- **其他支持HTTP请求的AI工具**
