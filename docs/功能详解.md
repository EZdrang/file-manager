# 资料管理系统 v3.2 - 功能详解

## 一、核心功能

### 1. 工作目录管理
- **工作目录模式**：选择一个文件夹作为主工作目录，直接读取文件系统
- **链接外部文件夹**：可以链接其他位置的文件夹到侧边栏，方便统一管理
- **目录树**：VSCode风格的目录树，支持展开/折叠，带连线显示层级关系
- **自动收缩**：导航时自动收缩未浏览的文件夹（可在设置中关闭）

### 2. 文件浏览与预览
- **单击预览**：单击文件即可在右侧面板预览内容
- **双击打开**：双击文件用系统默认程序打开
- **多标签页**：支持在新标签页中打开文件夹（右键菜单）
- **面包屑导航**：顶部显示当前路径，可点击跳转
- **返回按钮**：快速返回上一层目录

### 3. 文件类型支持
支持预览的文件类型：
- **文本文件**：txt, md, log, csv, json, xml, yaml, ini, cfg
- **代码文件**：py, js, ts, java, c, cpp, h, go, rs, rb, php, html, css, vue, svelte
- **图片文件**：png, jpg, jpeg, gif, bmp, webp, svg, ico, tiff（支持缩放、旋转）
- **PDF文档**：支持直接预览，可缩放
- **Word文档**：docx 格式直接渲染预览
- **音频文件**：mp3, wav, flac, aac, ogg 等直接播放
- **HTML文件**：实时渲染网页效果
- **Markdown**：支持完整 Markdown 渲染预览
- **脚本文件**：bat/sh/ps1 代码高亮预览
- **压缩包**：ZIP/RAR/7Z/TAR/GZ 内容列表预览
- **其他**：显示文件类型图标和语言标识

### 4. 文件类型图标系统
每种文件类型都有专属emoji图标：
- 🐍 Python | 📜 JavaScript | ☕ Java | ⚙️ C/C++ | 🌐 HTML | 🎨 CSS
- 📄 文档 | 📊 表格 | 📦 压缩包 | 🎵 音频 | 🎬 视频 | 🖼️ 图片

### 5. 标签与备注系统
- **彩色标签**：为文件添加彩色标签，支持多个标签（逗号分隔）
- **备注功能**：为文件添加文字备注
- **标签颜色**：基于标签名自动生成15种颜色之一
- **实时显示**：标签在文件列表中实时显示

### 6. 智能搜索
#### 基础搜索
- 输入关键词搜索文件名
- 支持模糊匹配

#### 高级筛选语法
| 语法 | 功能 | 示例 |
|------|------|------|
| `#标签` | 按标签筛选 | `#重要` |
| `!扩展名` | 按文件类型筛选 | `!py` `!image` `!code` |
| `@备注` | 按备注内容筛选 | `@待处理` |
| `yaml:key` | YAML标签筛选 | `yaml:category` |
| `-#标签` | 排除标签 | `-#废弃` |
| `-!扩展名` | 排除类型 | `-!exe` |
| `空格` | AND逻辑（同时满足） | `#重要 !py` |
| `|` | OR逻辑（满足其一） | `#重要|#紧急` |

#### 快捷类型分组
- `!image` → png, jpg, gif, bmp, webp, svg, ico
- `!code` → py, js, ts, java, c, cpp, go, rs, rb, php
- `!doc` → doc, docx, pdf, txt, md

#### 中文符号支持
全角符号自动转换：`＃`→`#` `！`→`!` `＠`→`@` `｜`→`|` `：`→`:`

#### 路径跳转
在搜索框输入Windows路径（如 `C:\Users\admin`）自动跳转到该目录

#### 文件右键菜单

- 复制路径/文件名
- 备份到同目录

- 重命名/删除

#### 文件夹右键菜单

- 重命名/删除

#### 空白处右键菜单
- 刷新
- 新建文件夹
- 从资源管理器粘贴
- 在资源管理器中打开

#### 预览区右键菜单
- 复制选中文本

#### 标签/备注输入框右键菜单
- 剪切/复制/粘贴
- 全选/删除

### 8. 文件操作
- **复制/剪切/粘贴**：支持应用内和系统剪切板
- **备份**：一键备份文件到同目录（自动添加时间戳）
- **重命名**：支持重命名文件和文件夹，支持正则批量重命名
- **删除**：删除文件或文件夹（移入回收站）
- **拖拽**：从App拖文件到资源管理器，或拖到文件夹直接移动
- **文件夹拖拽移动**：拖文件到文件夹图标上直接移动
- **跨栏拖拽**：多栏模式下，拖文件到另一个面板实现跨栏移动/复制
- **撤销/重做**：Ctrl+Z 撤销，Ctrl+Y 重做，支持删除/移动/重命名操作
- **键盘快捷键**：
  - `Ctrl+C` 复制
  - `Ctrl+X` 剪切
  - `Ctrl+V` 粘贴
  - `Ctrl+A` 全选
  - `Ctrl+N` 新建文件夹
  - `Ctrl+Z` 撤销
  - `Ctrl+Y` 重做
  - `F2` 重命名
  - `F5` 刷新
  - `Delete` 删除
  - `Backspace` 返回上一层
  - `Enter` 打开选中项
  - `Esc` 关闭面板

### 9. 主题系统
- **暗色主题**：深色渐变背景，护眼
- **亮色主题**：浅色背景，清晰
- **一键切换**：侧边栏底部快速切换
- **记住选择**：主题设置自动保存

### 10. 多窗口模式
- **标签页**：支持在新标签页中打开文件夹
- **多窗口**：支持在新窗口中打开文件夹
- **设置开关**：可在设置中开启/关闭

### 11. 多栏模式（v3.1）
- **1栏/2栏/3栏**：工具栏按钮切换
- **跨栏拖拽**：拖文件到另一个面板，移动到该面板当前目录
- **Ctrl+拖拽**：跨栏复制文件
- **每栏独立状态**：排序、过滤、视图模式各自独立

### 12. 文件过滤器（v3.1）
- **类型过滤**：图片/视频/音频/文档/代码/压缩包
- **大小过滤**：<100KB / 100KB-1MB / 1MB-10MB / 10MB-100MB / >100MB
- **时间过滤**：今天/最近一周/最近一月/最近一年
- **清除按钮**：一键清除所有过滤条件

### 13. 分类视图（v3.1）
- **列表视图**：默认视图
- **分类视图**：按文件类型自动分组（图片/视频/音频/文档/代码/压缩包/其他）
- **看板视图**：按标签分组展示，支持拖拽标签

### 14. 批量操作（v3.1）
- **多选**：Ctrl+点击选择多个，Shift+点击范围选择
- **批量打标签**：为多个文件统一添加标签
- **批量删标签**：为多个文件统一删除标签
- **批量删备注**：为多个文件统一删除备注
- **批量重命名**：支持正则表达式替换
- **批量移动**：将多个文件移动到指定目录
- **批量删除**：删除多个选中的文件

### 15. 知识笔记系统（v3.1）
- **文件关联笔记**：右键文件可关联 Markdown 笔记
- **笔记编辑器**：支持 Markdown 语法，字号可调
- **笔记列表**：查看所有有笔记的文件
- **导出笔记**：支持导出单篇或全部笔记为 Markdown 文件
- **笔记菜单**：顶部菜单栏"笔记"菜单

### 16. EXIF 信息与自动打标签（v3.1）
- **EXIF 显示**：图片预览时自动显示拍摄设备、镜头、焦距、光圈等

### 17. 文件夹备注（v3.1）

- **右键操作**：目录树右键可设置/修改文件夹备注

### 18. 收藏夹与最近文件（v3.1）

## 三、REST API服务

### 概述
内置HTTP API服务器，端口默认5000，供AI Agent调用。

### API端点

| 方法 | 路径 | 功能 |
|------|------|------|
| GET | `/api/health` | 健康检查 |
| GET | `/api/workspace` | 工作目录信息 |
| GET | `/api/files` | 列出文件 |
| GET | `/api/files/*` | 读取文件内容 |
| GET | `/api/stats` | 统计信息 |
| GET | `/api/docs` | API文档 |
| POST | `/api/search` | 搜索文件 |
| POST | `/api/meta` | 获取标签/备注 |
| PUT | `/api/meta` | 保存标签/备注 |
| POST | `/api/create-folder` | 创建文件夹 |
| POST | `/api/delete` | 删除文件 |
| POST | `/api/rename` | 重命名 |

### 调用示例

```bash
# 健康检查
curl http://127.0.0.1:5000/api/health

# 获取工作目录信息
curl http://127.0.0.1:5000/api/workspace

# 列出文件
curl http://127.0.0.1:5000/api/files?path=src&depth=1

# 搜索文件
curl -X POST http://127.0.0.1:5000/api/search \
  -H "Content-Type: application/json" \
  -d '{"keyword": "main.js"}'

# 读取文件内容
curl http://127.0.0.1:5000/api/files/src/main.js

# 给文件打标签
curl -X PUT http://127.0.0.1:5000/api/meta \
  -H "Content-Type: application/json" \
  -d '{"path": "src/main.js", "tags": "重要,核心", "notes": "主入口文件"}'

# 获取文件标签
curl -X POST http://127.0.0.1:5000/api/meta \
  -H "Content-Type: application/json" \
  -d '{"path": "src/main.js"}'

# 创建文件夹
curl -X POST http://127.0.0.1:5000/api/create-folder \
  -H "Content-Type: application/json" \
  -d '{"name": "new-folder"}'

# 删除文件
curl -X POST http://127.0.0.1:5000/api/delete \
  -H "Content-Type: application/json" \
  -d '{"path": "old-file.txt"}'

# 重命名
curl -X POST http://127.0.0.1:5000/api/rename \
  -H "Content-Type: application/json" \
  -d '{"path": "old-name.txt", "newName": "new-name.txt"}'
```

- **其他支持HTTP请求的AI工具**
